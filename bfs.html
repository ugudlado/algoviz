Pseudocode
queue = [start] visited = {start} while queue is not empty: node = queue.dequeue() if node == goal: return reconstruct_path() for neighbor of node: if neighbor not in visited: visited.add(neighbor) queue.enqueue(neighbor)
Real-world Analogy

Ripples in a pond — drop a stone at the start point; waves spread outward one layer at a time, reaching every cell at the shortest distance first. BFS explores all neighbors before moving deeper.
