Pseudocode
insert(word): for i from 1 to k: idx = hash_i(word) mod m bits[idx] = 1 query(word): for i from 1 to k: idx = hash_i(word) mod m if bits[idx] == 0: return "definitely not" return "probably yes"
Real-world Analogy

A password breach checker — a website checks if your password has been seen in known data breaches. Instead of storing billions of passwords, it uses a bloom filter: fast “definitely not compromised” answers, with rare false alarms.
