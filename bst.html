Pseudocode
insert(node, value): if node is null: return new Node(value) if value < node.val: node.left = insert(node.left, value) else: node.right = insert(node.right, value) return node inorder(node): if node is null: return inorder(node.left) visit(node) inorder(node.right)
Real-world Analogy

A library card catalog — each card directs you left (earlier in the alphabet) or right (later). To find a book, you follow the branches until you reach it. Traversals are like reading every card in a specific order.
