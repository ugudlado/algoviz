Context

The Problem

You want to explore every reachable node in a graph — perhaps to detect cycles, find connected components, or solve a maze.

The Approach

Go as deep as possible down one path before backtracking. Use a stack (or recursion). Mark nodes visited to avoid infinite loops.

When To Use

Cycle detection, topological sort, maze solving, finding connected components. Uses less memory than BFS for deep graphs.

Pseudocode
stack = [start] visited = {} while stack is not empty: node = stack.pop() if node == goal: return reconstruct_path() if node not in visited: visited.add(node) for neighbor of node: stack.push(neighbor)
Real-world Analogy

Exploring a maze — you walk as far as you can down one corridor before hitting a dead end, then backtrack to the last junction and try the next corridor. DFS dives deep before exploring wide.
