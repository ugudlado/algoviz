Context

The Problem

You're packing a hiking bag with a weight limit. Each item has a weight and value. How do you maximize value without exceeding the weight limit?

The Approach

Build a table: for each item and each possible weight limit, decide: skip this item (same value as without it) or include it (add its value + best for remaining weight).

When To Use

Resource allocation under constraints. Budget optimization, cargo loading, project selection, cutting stock problems.

-- Optimal Value
Pseudocode
for i from 1 to n: for w from 0 to W: if wt[i] > w: dp[i][w] = dp[i-1][w] else: dp[i][w] = max( dp[i-1][w], dp[i-1][w-wt[i]] + val[i] )
Real-world Analogy

Packing a suitcase for a flight — you have a weight limit and must choose which items to bring to maximize value. Each item is all-or-nothing: you can't bring half a laptop.
