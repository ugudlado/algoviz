Pseudocode
sort edges by weight ascending mst = [] for each edge (u, v, w): if find(u) != find(v): union(u, v) mst.add(edge) else: reject (cycle) return mst
Real-world Analogy

Connecting villages with roads — list all possible roads by cost, cheapest first. Build each road only if it connects a village not yet reachable — skip any road that would create a loop. The result is the cheapest network connecting everyone.
