Context

The Problem

Your spell-checker receives 'recieve'. How does it know the closest correct word is 'receive'? How do you measure how different two strings are?

The Approach

Count the minimum edits (insert, delete, substitute) to transform one string into another. Fill a table where each cell is the edit distance for substrings.

When To Use

Spell checking, DNA sequence alignment, fuzzy search, plagiarism detection, autocorrect.

-- Edit Distance
Pseudocode
for i from 1 to m: for j from 1 to n: if source[i] == target[j]: dp[i][j] = dp[i-1][j-1] else: dp[i][j] = 1 + min( dp[i-1][j-1], // substitute dp[i][j-1], // insert dp[i-1][j] // delete )
Real-world Analogy

Autocorrect on your phone — when you mistype a word, the keyboard calculates the edit distance to every dictionary word and suggests the closest matches. Fewer edits (insert, delete, substitute) means a better suggestion.
