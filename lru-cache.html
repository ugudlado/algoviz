Pseudocode
get(key): if key not in map: return -1 move node to front of list return node.value put(key, value): if key in map: update node.value move node to front else: if size == capacity: evict tail (LRU) add new node to front map[key] = node
Real-world Analogy

Browser tab management — your browser keeps recently visited pages in memory. When memory is full, the least-recently-used tab gets evicted first. Revisiting a tab moves it to the front of the “keep alive” list.
