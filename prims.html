Context

The Problem

You're laying fiber optic cable to connect 50 cities. Each potential cable segment has a cost. How do you connect all cities with minimum total cable?

The Approach

Grow a tree one edge at a time. Always add the cheapest edge that connects a new city to the already-connected group. The result spans all cities optimally.

When To Use

Minimum Spanning Tree for dense graphs. Network design (telecommunications, electrical grids, water pipes). Cluster analysis.

Pseudocode
mst = {}, key[start] = 0 pq = [(0, start)] while pq is not empty: (w, u) = pq.extractMin() if u in mst: continue mst.add(u) for (v, weight) in adj[u]: if v not in mst and weight < key[v]: key[v] = weight parent[v] = u pq.insert((weight, v))
Real-world Analogy

Expanding a power grid — start at one city and always extend the grid to the nearest unconnected city using the cheapest cable. Repeat until all cities are connected with minimum total cable cost.
