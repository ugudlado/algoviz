Pseudocode (Fixed Window)
sum = sum(arr[0..k-1]) maxSum = sum for i from k to n-1: sum += arr[i] - arr[i-k] maxSum = max(maxSum, sum) return maxSum
Pseudocode (Variable Window)
left = 0, best = 0 for right from 0 to n-1: add arr[right] to window while window invalid: remove arr[left], left++ best = max(best, right-left+1) return best
Real-world Analogy

A train window view — as the train moves, new scenery enters one side of the window while old scenery exits the other. You always see exactly k items at once (fixed) or expand and shrink the window to find the best view (variable).
