Context

The Problem

You're building autocomplete for a search box. Checking every word in a dictionary for each prefix is too slow. How do you find all words with prefix 'alg' instantly?

The Approach

Store words character by character in a tree. Each node represents one character. All words sharing a prefix share the same path from root.

When To Use

Autocomplete, spell checking, IP routing (longest prefix match), word games, dictionary compression.

Pseudocode
insert(word): node = root for each char c in word: if c not in node.children: node.children[c] = new Node node = node.children[c] node.isEnd = true
Real-world Analogy

Phone book autocomplete — as you type each letter, the system narrows down matches by following a branching path. Shared prefixes share the same path, so "cat" and "car" diverge only at the third character.
